# Interview Project

Front End Intervew Project

## Task Summary

Objective: Build a browser application to retrieve items from your cart.

This should be built using:

- React Typescript
- Functional Component with Hooks

## Run Locally

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
